# 🌟 Overview

**Ludo Game** is a complete browser-based implementation of the classic board game loved by millions around the world.

Built entirely with **Vanilla JavaScript**, this project offers smooth gameplay, animated interactions, responsive design, and all the essential Ludo mechanics without relying on external libraries.

Simply open the game in your browser and enjoy playing with friends and family.

# 🎮 How to Play

1. Roll the dice.
2. Get a **6** to move a token out of the base.
3. Select the token you want to move.
4. Capture opponents by landing on their cells.
5. Reach the center safely.
6. Bring all **4 tokens home** to win the game.

---

# 📖 Game Rules

| Rule            | Description                              |
| --------------- | ---------------------------------------- |
| 👥 Players      | 2, 3, or 4 Players                       |
| 🎲 Enter Board  | Roll a 6 to release a token              |
| 🔁 Bonus Turn   | Roll a 6 to play again                   |
| ⚔️ Capture      | Land on opponent's token to send it back |
| ⭐ Safe Spots    | Tokens cannot be captured here           |
| 🏠 Home Stretch | Only your tokens can enter               |
| 🏆 Winning      | Bring all 4 tokens to the center         |
